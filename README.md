# RamCalculator

This project mainly started because i had to always open a website to try converting 1GB into 1024mb and doing it for 12GB isnt that easy (atleast not for me). So i wanted a better solution then allways opening up a website or calculating.

The easyest way to do it in ur head is to calculate it like this:

12GB = 1000mb * 12 + 24mb * 12
12GB = 12000mb + 288mb
12GB = 12288mb

This is duable in the head but just not efficent.
So i build a "calculator" that does it for u!

## Install
I highly recommend using the code with python and installing Qt6.
There is a windows exe but it looks not how it should for somereason and i cant get it to compile with everything for linux for the life of me.

Input a number and pick a Size(TB,GB,MB,KB,B) and it shows in Hex or decimal.
